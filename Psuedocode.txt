Laboratory Activity 1: Add Two Numbers
Psuedocode:
START
INPUT num1
INPUT num2
PROCESS sum = num1 + num2
DISPLAY sum
END

Laboratory Activity 2: Compute the average of Three Quiz Scores
Psuedocode:
START
INPUT score1
INPUT score2
INPUT score3
PROCESS average=(score1+score2+score3)
DISPLAY average
END

Laboratory Activity 3: Determine if a Number is Even or Odd
Psuedocode:
START
INPUT number
IF number MOD 2 = 0 THEN
   Display "Even"
ELSE
   Display "Odd"
END IF
END
